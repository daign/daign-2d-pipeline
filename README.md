# daign-2d-pipeline

[![NPM package][npm]][npm-url]

Two dimensional graphics pipeline.

## Installation

```sh
npm install @daign/2d-pipeline --save
```

## Scripts

#### Build

    npm run build

#### Run lint analysis

    npm run lint

#### Run unit tests with code coverage

    npm run test
